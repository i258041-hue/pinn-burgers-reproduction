# Physics-Informed Neural Networks (PINNs)

This repository contains the implementation, reproduction, and experimentation of the paper:

**Physics-informed neural networks: A deep learning framework for solving forward and inverse problems involving nonlinear partial differential equations**  
Authors: M. Raissi, P. Perdikaris, G.E. Karniadakis

## Repository Structure

- `train.py` → Training script
- `inference.py` → Model inference script
- `config.yaml` → Configuration file
- `src/` → Source code modules
- `results/` → Metrics and training logs
- `checkpoints/` → Saved model weights
- `notebooks/` → Demonstration notebooks
- `data/` → Sample dataset

## Tasks Completed

1. Paper Understanding
2. Reproduction of Results
3. Experimentation and Expansion

## Requirements

Install dependencies:

```bash
pip install -r requirements.txt
```

## Running Training

```bash
python train.py
```

## Running Inference

```bash
python inference.py
```
